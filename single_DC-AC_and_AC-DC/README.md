# 单相DC-AC与AC-DC — 单相逆变/整流项目

## 项目简介
单相并网/离网逆变器与整流拓扑的立创EDA工程，基于STM32G474。

## 目录结构
| 目录/文件 | 说明 |
|-----------|------|
| `SinglePhase_GridTie_OffGrid_Inverter_G474/` | STM32G474 主控工程 |
| `单相逆变离并网和整流拓扑/` | 逆变整流拓扑 EDA 工程 |
| `单相逆变并网/论文/` | 并网论文（已移至 GitHub Release） |
| `单相逆变离网/论文/` | 离网论文（已移至 GitHub Release） |
| `单相整流/论文/` | 整流论文（已移至 GitHub Release） |
