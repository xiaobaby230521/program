# 单相DC-AC与AC-DC — 单相逆变/整流项目

## 项目简介
单相并网/离网逆变器与整流拓扑的立创EDA工程，基于STM32G474。

## 目录结构
| 目录/文件 | 说明 |
|-----------|------|
| `SinglePhase_GridTie_OffGrid_Inverter_G474/` | STM32G474 Keil MDK 工程 |
| `SinglePhase_GridTie_OffGrid_Inverter_G474/Core/` | CMSIS与启动文件 |
| `SinglePhase_GridTie_OffGrid_Inverter_G474/Drivers/` | HAL外设库 |
| `SinglePhase_GridTie_OffGrid_Inverter_G474/MDK-ARM/` | Keil工程文件（.uvprojx/.uvoptx） |
| `SinglePhase_GridTie_OffGrid_Inverter_G474.ioc` | STM32CubeMX 工程配置 |
| `SinglePhase_GridTie_OffGrid_Inverter_G474.zip` | 工程压缩包 |
| `单相逆变离并网和整流拓扑/` | 逆变整流拓扑 EDA 工程 |
| `单相逆变离并网和整流拓扑/*.eprj` | 立创EDA 工程文件 |
| `单相逆变离并网和整流拓扑/*.epro` | 立创EDA 工程配置 |
| `单相逆变并网/` | 单相并网逆变仿真与论文 |
| `单相逆变并网/仿真/` | Simulink模型与MATLAB脚本 |
| `单相逆变并网/论文/` | 并网逆变论文（13篇PDF） |
| `单相逆变离网/` | 单相离网逆变仿真与论文 |
| `单相逆变离网/仿真文件/` | Simulink模型与MATLAB脚本 |
| `单相逆变离网/论文/` | 离网逆变论文（14篇PDF） |
| `单相整流/` | 单相PWM整流仿真与论文 |
| `单相整流/仿真/` | Simulink模型 |
| `单相整流/论文/` | 整流论文（25篇PDF） |
