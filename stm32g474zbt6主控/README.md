# STM32G474 主控 — 微控制器PCB设计

## 项目简介
STM32G474RBT6 主控板的立创EDA工程，含评估板参考设计。

## 目录结构
| 目录/文件 | 说明 |
|-----------|------|
| `主控G4.eprj` | 主控板 EDA 工程 |
| `主控G4_backup/` | 主控板历史备份（v146_2026-03-22） |
| `STM32G474RBT6直插主控/` | 直插式主控板 EDA 工程 |
| `STM32G474RBT6直插主控/STM32G474RBT6直插主控.eprj` | 直插版EDA工程文件 |
| `STM32G474RBT6直插主控/STM32G474RBT6直插主控板.epro` | 直插版EDA工程配置 |
| `STM32G474RBT6直插主控/STM32G474RBT6直插主控_backup/` | 直插版历史备份（v87_2026-04-15） |
| `STM32G474RBT6_规格书_WJ50867.PDF` | 芯片规格书 |
| `stm32g474官方评估板pcb.pdf` | 官方评估板PCB参考 |
