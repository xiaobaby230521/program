# 工科赛 — 电机电压电流采样

## 项目简介
工科赛电机电压电流采样电路的立创EDA工程。

## 目录结构
| 目录/文件 | 说明 |
|-----------|------|
| `STM32G0_GKS校赛定稿/` | STM32G0 Keil MDK 工程 |
| `STM32G0_GKS校赛定稿/STM32G0_GKS校赛定稿/` | 工程根目录 |
| `STM32G0_GKS校赛定稿/.../Core/` | CMSIS与启动文件 |
| `STM32G0_GKS校赛定稿/.../Drivers/` | HAL外设库（STM32G0xx） |
| `STM32G0_GKS校赛定稿/.../MDK-ARM/` | Keil工程（.uvprojx / .uvoptx） |
| `STM32G0_GKS校赛定稿/.../STM32G0_GetADC.ioc` | STM32CubeMX 工程配置 |
| `工科赛(1)(2)(1).eprj` | 采样电路 EDA 工程 |
