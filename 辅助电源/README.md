# 辅助电源 — 辅助电源模块设计

## 项目简介
辅助电源模块的立创EDA工程，含直插式和数字功率隔离方案。

## 目录结构
| 目录/文件 | 方案/说明 |
|-----------|------|
| `辅助电源.eprj` | 辅助电源主工程 |
| `辅助电源.epro` | 主工程配置 |
| `辅助电源_backup/` | 主工程历史备份（v73/v76/v79） |
| `直插辅助电源/` | 直插式方案（3.3V/5V/12V） |
| `直插辅助电源/直插辅助电源3.3，5，12.eprj` | 直插式工程文件 |
| `直插辅助电源/直插辅助电源3.3，5，12_backup/` | 直插版备份（v65/v67） |
| `直插数字功率隔离辅助电源/` | 数字功率隔离方案 |
| `直插数字功率隔离辅助电源/*.eprj` | 数字功率隔离EDA工程 |
| `直插数字功率隔离辅助电源/*.epro` | 数字功率隔离工程配置 |
| `直插数字功率隔离辅助电源_backup/` | 数字功率隔离备份（v41/v42） |
| `直插稳压隔离辅助电源/` | 稳压隔离方案 |
| `直插稳压隔离辅助电源/*.eprj` | 稳压隔离EDA工程 |
| `直插稳压隔离辅助电源/*.epro` | 稳压隔离工程配置 |
