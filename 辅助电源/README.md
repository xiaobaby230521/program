# 辅助电源 — 辅助电源模块设计

## 项目简介
辅助电源模块的立创EDA工程，含直插式和数字功率隔离方案。

## 目录结构
| 目录/文件 | 说明 |
|-----------|------|
| `辅助电源.eprj` | 辅助电源主工程 |
| `辅助电源_backup/` | 历史备份压缩包 |
| `直插辅助电源/` | 直插式辅助电源（3.3V/5V/12V） |
| `直插数字功率隔离辅助电源/` | 数字功率隔离方案 |
| `直插稳压隔离辅助电源/` | 稳压隔离辅助电源 |
